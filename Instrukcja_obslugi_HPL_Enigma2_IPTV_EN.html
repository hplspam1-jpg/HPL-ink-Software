2. Startup and initial configuration

Requirements

  • 64-bit Windows 11.
  • An Enigma2 receiver with OpenWebif enabled, on the same network as the PC.
  • An M3U/M3U8 playlist when IPTV is required.
  • 64-bit VLC when the external VLC button or functions requesting its location are used.
  • A TMDb API key for movie/series details and posters. PRO + ULTRA Create or sign in to a TMDb account, open Account settings → API on a desktop browser, accept the terms and request a developer key. Official guide: TMDb Getting Started.

Portable operation

  1. Extract the complete program folder. Do not move the EXE away from its libraries and subfolders.
  2. Run HplEnigma2IptvWinUI.exe.
  3. If the application folder is writable, data is stored next to the EXE. Otherwise, the program uses %LocalAppData%\HplEnigma2Iptv.

Connecting to Enigma2

1. IPEnter the receiver address on the local network.
2. USER/PASSEnter OpenWebif credentials when authentication is enabled.
3. PORT/WEBTypical values are stream 8001 and OpenWebif 80.
4. SaveSave the configuration and select Bouquets.

Locate VLC selects the VLC executable. VLC opens the current stream in the external player and is useful for diagnostics.

4. Enigma2

Loading and playing channels

  1. Complete the connection fields and select Save.
  2. Select Bouquets. Red bold entries represent bouquets/folders.
  3. Select a bouquet and double-click a channel.
  4. Current and next EPG appears below the video when the receiver supplies it.

The channel context menu provides Play, Add to favourites, Show EPG and Copy URL. The bouquet context menu loads the bouquet or copies its service reference.

Remote control

Remote opens the virtual Enigma2 remote, including navigation, number, media and REC keys. Commands are sent through OpenWebif.

Enigma2 reconnect PRO + ULTRA

After a stream failure, the program attempts to resume playback. Manual Stop prevents reconnect for the stopped channel. The function can be disabled in Settings.

Channel picons ULTRA ONLY

Picons appear in the E2 list and above the player. They can be placed in Picon\Enigma2 next to the EXE or downloaded from the receiver with Picon manager.

Signal strength ULTRA ONLY

Select Tools → Signal strength. The window shows SNR, SNR dB, AGC, BER and lock state for the current E2 service, refreshing every two seconds.

5. IPTV and M3U playlists

Loading a playlist

  1. Select M3U and open an .m3u or .m3u8 file.
  2. Groups appear in the IPTV panel. All displays all live TV channels.
  3. Select a group and double-click a channel.
  4. IPTV opens in a separate window and can run concurrently with Enigma2.
Privacy: loading or browsing a playlist does not open every stream URL. A provider connection is made when a selected item is played or a referenced resource such as a logo is fetched.

M3U editor

M3U Ed. edits channel name, group, URL and logo. Auto logo assigns available graphics. PRO and ULTRA can rebuild the editable list from the currently loaded playlist. PRO + ULTRA

Custom group order PRO + ULTRA

Right-click a group and choose Move up or Move down. All remains pinned at the top.

IPTV player

The window includes Pause/Play, Stop, Video, Cinema, volume control, current EPG and a channel logo when available. Closing IPTV does not close the main program.

IPTV reconnect ULTRA ONLY

Live channels receive up to three reconnect attempts at five-second intervals. VOD movies and series are not restarted after their natural end.

6. IPTV Movies and IPTV Series PRO + ULTRA

Requirements

  • The M3U playlist should identify media through paths or metadata such as /movie/ and /series/.
  • Enter a TMDb API key in Settings.
  • The provider account must allow the selected movie or episode to play.

IPTV Movies

  1. Load M3U and select IPTV Movies.
  2. Open a movie folder to display the poster wall and search box.
  3. Select Download posters to cache missing images.
  4. Open a movie to see poster, year, rating, director and overview, then select Play.

IPTV Series

Items are organised as series → season → episode. Select an episode and start playback.

Resume VOD ULTRA ONLY

ULTRA stores the position of movies, series and local recordings. The next session can continue or use Start over.

7. EPG

EPG XML

EPG XML loads a local XMLTV file. Gzip-compressed data is also supported. Entries are matched to IPTV channel names.

Download EPG

Download EPG opens the download window. Enter the source address and destination, normally epg.xml. “Valid until” reflects the actual final programme in the data.

Bouquet EPG

Select an Enigma2 bouquet first, then Bouquet EPG. The window displays programmes for channels in that bouquet.

Record from EPG PRO + ULTRA

Use Record for the selected programme. Depending on the command and window, the program creates an Enigma2 timer or starts a local recording.

EPG reminders PRO + ULTRA

Enable reminders in Settings, select 5-15 minutes, optionally restrict to Favourites and enter comma-separated keywords.

8. Recording and timeshift

Record and recording schedule PRO + ULTRA

Record saves the current Enigma2 stream in Recorded next to the EXE. Press Record again or Stop to finish. The schedule window reads Enigma2 timers and requires receiver confirmation when deleting one.

Recording library ULTRA ONLY

Tools → Recording library lists TS, MP4, MKV and AVI files in Recorded. It supports playback, rename, delete, size and date information.

Recording monitor ULTRA ONLY

Displays elapsed time, file size, write speed and free disk space.

Timeshift ULTRA ONLY

  1. Start an Enigma2 channel and select Timeshift.
  2. The live stream is buffered to a temporary file.
  3. Pause and seek within the available buffer.
  4. Return to LIVE or stop timeshift. The temporary file is removed.

9. Playback and video modes

VIDEO

In the main window, VIDEO enlarges the Enigma2 video area and hides control panels. ULTRA requires an active E2 channel and displays a message otherwise. ULTRA ONLY: no-source guard

Cinema / F11

Enters full screen. Esc returns to the GUI and restores the default window size.

PiP

Opens a small MiniPlayer window. Closing PiP does not close the application.

Tracks

Displays VLC audio and subtitle tracks available for the current item. The active track is highlighted.

Audio

Audio-only mode hides video while keeping sound, for example for radio services.

Stop

Stops Enigma2 in the main window. Stop in IPTV affects only the IPTV window.

10. Canal+ Online, DLNA, Chromecast and M3U TV

Canal+ Online and Polsat Box

Opens the integrated HPL Streaming module. Run opens operator pages in the original Microsoft Edge browser and displays My List. Authentication takes place on the operator website.

DLNA

Select DLNA, set the search duration and choose Search devices. VPN, Wi-Fi client isolation and firewall rules can block SSDP discovery.

Chromecast

Select Chromecast, the target device and source. The IPTV context menu also provides Send to Chromecast. The receiving device must be able to reach the source URL.

M3U TV

Starts a local HTTP server for the loaded playlist. Load M3U first. The window shows server URL, status and request count.

12. Settings - every option from top to bottom PRO + ULTRA

Select Settings in the upper-right corner. The controls appear in the following order from the top. Scroll vertically and select Save settings when finished.

  1. Language and theme.
    • Interface language - select Polski (PL) or English (EN). Restart the program to apply it.
    • GUI theme - Black (classic) or Futuristic HUD. Restart to apply the new appearance.
  2. Startup profile.
    • Auto-connect to receiver - automatically uses the active Enigma2 receiver after startup.
    • Play last channel - attempts to resume the most recently watched item.
    • Window at startup - None, Streaming Canal+/Polsat, Remote control or Download EPG.
    • Network quality preset - LAN uses the smallest buffer, WiFi a medium buffer and Mobile the largest, safest buffer for weaker connections.
    • Network/live buffer (ms) - range 150-10000 ms. A larger value usually reduces interruptions but increases startup time and live delay.
  3. EPG reminders.
    • Enabled - turns programme reminders on.
    • Minutes before the programme - select 5 to 15 minutes.
    • Favourites only - limits reminders to favourite channels.
    • EPG keywords - enter comma-separated phrases, for example movie, match, news. The programme title and channel name are checked.
  4. Parental lock.
    • Enabled - activates access control.
    • New PIN - enter at least four digits and select Set PIN.
    • Locked M3U groups - enter exact IPTV group names separated by commas.
    • Locked Enigma2 bouquets - enter exact E2 bouquet names separated by commas.
  5. Backup.
    • Backup .hplbackup - enter a password of at least four characters and choose where to save the encrypted configuration backup.
    • Restore backup - enter the password and select an .hplbackup file. A wrong password or damaged file displays an error instead of closing the program.
  6. Other.
    • Enigma2 video decoding - Auto, GPU or CPU. AUTO starts with GPU and switches the machine to CPU when audio is detected without video. ULTRA ONLY
    • Reset saved AUTO choice - removes the GPU/CPU decision stored for this PC so detection runs again. ULTRA ONLY
    • Audio-only mode (radio) - disables video and keeps audio.
    • Automatic Enigma2 reconnect - attempts to restart the current channel after a stream failure.
    • Automatic IPTV reconnect - up to three attempts every five seconds for live channels; VOD is excluded. ULTRA ONLY
    • TMDb API key - paste the v3 API key used by IPTV Movies/Series for details and posters. To obtain one, create or sign in to an account, open TMDb → Settings → API in a desktop browser, accept the terms and complete the short developer application. See the official TMDb guide. PRO + ULTRA
    • Enigma2 picon folder (FTP) - enter the remote path, for example /media/usb/pICON. ULTRA ONLY
    • Receiver FTP port - normally 21; change it only if the receiver FTP server uses another port. ULTRA ONLY
    • Timeshift buffer folder - choose a fast drive with plenty of free space. An empty field uses the default application data folder. ULTRA ONLY
    • Local channel logo folder - select a folder containing files named exactly like IPTV channels. ULTRA ONLY
  7. Save settings. The bottom button stores all changes and closes the window. Language and theme changes require a program restart.

Local channel logo folder ULTRA ONLY

A file name must match the channel, for example TVP 1 HD.png. Supported formats are PNG, JPG, JPEG, WEBP and ICO. Priority: M3U tvg-logo → EPG icon → local folder.

Picon names ULTRA ONLY

The file name is the E2 service reference with colons replaced by underscores, for example 1_0_19_290E_1EDC_71_820000_0_0_0.png. PNG, JPG and JPEG are supported.

15. Troubleshooting

Bouquets and EPG work, but Enigma2 has audio only

  1. Test the same stream with the VLC button.
  2. Check the stream port, usually 8001, and WEB port, usually 80.
  3. In PRO, use the alternative GPU/CPU build. PRO
  4. In ULTRA, select AUTO or CPU decoding and reset the saved AUTO choice. ULTRA ONLY
  5. Select a larger buffer preset when sound or video is interrupted.

Live IPTV works, but a movie does not

Test the URL in VLC, verify account validity and concurrent connection limits. VOD uses LibVLC while live channels may use another playback engine.

Posters are missing

Check the TMDb key, internet connection and cleaned title. Open the movie folder and select Download posters.

DLNA or Chromecast is not found

The PC and device must be on the same network. Temporarily disable VPN, check Wi-Fi client isolation and allow private-network firewall access.

Temporary directory warning

“Failed to remove temporary directory” usually comes from an installer or updater. Close processes using TEMP, restart Windows and delete only the specified unused temporary folder.

Settings are not stored next to the EXE

Move the program to a user-writable folder or check %LocalAppData%\HplEnigma2Iptv.
