2. Requirements and startup

Requirements

  • Windows 10 version 2004 (build 19041) or newer; Windows 11 and a 64-bit system are recommended.
  • The complete application directory. Do not copy and run HPLMusicWinUI.exe without its libraries, Assets folder and FFmpeg tools.
  • Write access to the application folder for the index, settings, radio stations and covers.
  • Internet access only for online covers, radio and Chromecast. INTERNET

First run

  1. Extract the complete application folder.
  2. Run HPLMusicWinUI.exe.
  3. On the first run, select a music folder and index it.
  4. On later runs, an existing hpl_music_index.sqlite is loaded automatically.
Computer authorization: a licensed build may enforce MAC-address authorization. If the application reports that the computer is not authorized, contact the application supplier.

4. Indexing the library

1. Select folderChoose the parent folder that contains the music.
2. IndexThe application scans it and every subfolder.
3. WaitStatus shows the processed and total file counts.
4. Select a branchClick a folder in the tree to display recordings.

Indexing reads title, artist, album, year, genre, duration, format, size and path. Inaccessible folders are skipped instead of stopping the full scan. The result is saved in a local SQLite database.

Extensions included in a scan

.aac .ac3 .aiff .dts .flac .m4a .mp3 .ogg .opus .wav .wma

Good practice: re-index after adding, removing, moving or renaming a significant number of files.

5. Browsing, searching and organizing

Folders and table

Click a folder in the left tree. The right-hand list shows files from that branch and its subfolders. Click Format, Title, Artist, Album, Year, Genre, Duration or Filename to sort; click the same header again to reverse the order.

Search

  1. Enter one or more terms.
  2. Press Enter or click Search.
  3. Every term must occur in the file data. The search covers title, artist, album, year, genre, filename, folder and full path.
  4. Optionally select an extension in Type. The type filter works alone or together with text.

Duplicates

Duplicates lists files with the same filename and the same size. The application never deletes them automatically. Compare the paths and remove an unwanted copy in Windows File Explorer.

Clear removes search text, the type filter and result mode, then restores the library tree.

6. Editing metadata

  1. Right-click a file.
  2. Select Edit metadata.
  3. Edit the title, artist, album, year or genre.
  4. Click Save.
Caution: saving changes the tags in the original audio file. Back up important files before bulk editing. Some formats or file variants do not support reliable tag writing; the application then displays an error.

7. Downloading covers INTERNET FOR SOURCE 2

  1. Click Download covers.
  2. Select an indexed folder.
  3. Choose a source:
    • Source 1: from files reads artwork embedded in audio tags.
    • Source 2: online for missing searches MusicBrainz and downloads artwork from Cover Art Archive.
  4. Wait for the found, skipped, missing and error summary.

Existing assignments are skipped. Images are saved in hpl_music_covers and mappings in hpl_music_covers.sqlite. An orange dot in the player means a real cover was found; no dot means the fallback image is shown.

8. Player

Starting playback

  1. Select a folder or search result set.
  2. Select a recording.
  3. Click the green PLAY button or double-click the row.

The visible list becomes the player playlist and the selected file becomes its starting item. The main window is hidden; Show GUI brings it back without closing the player.

ControlAction

AC3 and DTS FFPLAY

.ac3 and .dts are played through ffplay.exe. Duration may remain at 0:00, seeking and conventional pause are unavailable, and Play acts as stop/restart. When playback ends naturally, the player advances to the next item.

9. Equalizer and mini mode

Visualizer types

Choose Bars, Neon LED reflection, Rainbow segments or Soundwave neon. The color can be multicolor or a single purple, light purple, neon red, turquoise, cyan, blue, green, yellow, orange or white.

The selection is saved and used in full and mini player views. Radio keeps its own independent EQ type and color.

Mini mode

Click Minimize for a wide compact panel with track information, controls, artwork and EQ. Click GUI to restore the full player. Closing the player stops music and reopens the main window.

10. Chromecast LOCAL NETWORK

  1. Connect the computer and Chromecast to the same network.
  2. Click Chromecast; discovery takes several seconds.
  3. If several receivers are found, choose one and confirm.
  4. After the connection message, select a file and use the normal PLAY button.

MP3, WAV, FLAC, AAC and M4A are served directly. Other supported formats are transcoded to MP3 by the local ffmpeg.exe. The application starts a temporary HTTP server, so Windows Firewall may ask for private-network access.

Important: keep both the application and computer running. Guest Wi-Fi, client isolation, VPN or firewall rules can prevent discovery or block the stream.

The player supports pause, resume, seek, volume and automatic next-track behavior. Click Disconnect Cast in the main window to end the session.

11. Internet radio INTERNET

Playback

  1. Click Radio.
  2. Select a station.
  3. Click Play or double-click the row.
  4. Adjust volume and EQ independently from file playback.

When no user database exists, the application provides starter stations. Search filters name, country, genre and URL; the genre list narrows the result. A station may link to audio directly or to an M3U, M3U8 or PLS playlist; the application attempts to resolve its first stream URL.

Add and edit

Add station and Edit provide name, stream URL, country, genre and logo URL fields. Name and URL are required. Delete asks for confirmation and saves the change.

M3U import and export

  • Load M3U accepts .m3u and .m3u8. Entries with URLs already stored are skipped.
  • Export M3U saves the currently visible stations. If a filter is active, only that filtered set is exported.

Mini view and playlist

Minimize switches radio to a wide mini panel. Hide/Show playlist works in both views. Closing the radio window stops the stream.

12. Bluetooth and audio output

Bluetooth chooses the output device for both player and radio. It does not pair Bluetooth hardware.

  1. If the device is not paired, click Open Windows BT settings and pair it in Windows.
  2. Return and click Refresh.
  3. Select the device or Windows default device.
  4. Click Set as active or double-click the row.

The choice is saved and applied immediately to an open player and radio. If the device disappears, select the Windows default output and refresh the list.

16. Troubleshooting

The list is empty after startup

Select a valid folder and click Index. After completion, click a folder in the left tree. Loading an index intentionally leaves the file table empty until a folder or search result is selected.

The index or settings are not saved

Data is stored next to the EXE. Move the complete directory to a user-writable location outside Program Files, then index again. Do not run a copied EXE on its own.

A file is missing

Check that its extension is supported and re-index. A folder without access permission may have been skipped.

No online cover is found

Correct the artist, album or title, check the Internet connection and retry. Not every release has artwork in MusicBrainz/Cover Art Archive.

AC3 or DTS does not play

Ensure ffplay.exe is in the application folder. The program can also search PATH and several common locations, but the local copy is recommended.

Chromecast is not found or does not play

Check the shared network, disable client isolation and VPN, allow private-network access in Windows Firewall, and ensure ffmpeg.exe exists for formats that need transcoding.

Radio does not play

Check Internet access and the station URL. The provider may have changed or blocked the stream. Test another station, then edit or delete the broken entry.

A Bluetooth device is missing

Pair and power on the device in Windows, click Refresh, and choose its audio profile. The application does not pair devices itself.

The application closed after an error

Check hpl_music_winui_crash.log in the application directory and provide it to support with a description of the last action.
